Cómo funcionan las características

Las caracteristicas unifican el sistema de preferencias:

  • Mobiliario: Asigna caracteristicas al mobiliario para definir sus atributos (ej: "Primera Linea", "Sombra")
  • Reservas: El cliente indica que caracteristicas desea y el sistema sugiere mobiliario compatible
  • Matching: El algoritmo de sugerencias compara directamente las caracteristicas
