Como funcionan las preferencias

Las preferencias se asignan a clientes y se usan en el algoritmo de sugerencias de mobiliario:

  • Codigo: Identificador unico para referencia interna
  • Mapea a: Caracteristica de mobiliario que coincide con esta preferencia
  • Cuando un cliente tiene una preferencia, el sistema sugiere mobiliario con caracteristicas coincidentes
